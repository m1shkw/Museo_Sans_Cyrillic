# РОВНЫЙ ШРИФТ ДЛЯ
РОВНОЙ ТИПОГРАФИКИ #

Museo Sans Cyrillic — это универсальный шрифт без засечек от студии exljbris font foundry, основанный на геометрическом шрифте Museo

Он был специально разработан для кириллических языков и поддерживает множество других идиом благодаря обширному набору символов

# ОПИСАНИЕ ШРИФТА

MUSEO

Museo-брекеты — бренд типографических брекетов. Стильное и удобное решение для кривозубиков. Набор глифов универсального шрифта «museo sans cyrl», делает зубки ровными, словно идеальная строка.
